# 📊 Superstore Sales Performance Dashboard

## 🔍 Overview
This project analyzes retail sales performance using an interactive Excel dashboard built with Power Query and Power Pivot.

The goal was to transform raw transactional data into meaningful insights to support business decision-making.

---

## 📌 Key Features
- KPI tracking (Total Sales, Profit, Orders, Return Rate, Profit Margin)
- Sales trend analysis over time
- Profit breakdown by product category
- Regional & managerial performance analysis
- Return rate analysis by category
- Interactive filtering using slicers (Region, Category, Segment)

---

## 🛠 Tools Used
- Microsoft Excel  
- Power Query (Data Cleaning & Transformation)  
- Power Pivot (Data Modeling & Measures)  
- Pivot Tables & Charts  

---

## 📈 Key Insights
- Technology is the most profitable product category  
- Central region generates the highest sales  
- Manager Chris leads in sales performance  
- Furniture has the highest return rate  
- Sales show seasonal fluctuations over time
